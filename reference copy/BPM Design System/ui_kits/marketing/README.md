# BPM Marketing Site UI kit

The public-facing marketing site. Cool, sober, institutional — closer to BlackRock's Aladdin landing or a Bain practice page than a SaaS launch. The hero is a single bilingual statement; the rest of the page proves the assertion with capability blocks, data, and named coverage.

## Files

| File | Purpose |
| --- | --- |
| `index.html` | Marketing homepage — hero, capability grid, stat strip, coverage, footer |
| `NavBar.jsx` | Sticky top bar — wordmark, three primary links, locale, "Request access" CTA |
| `Hero.jsx` | Bilingual hero with serif headline, paragraph, two CTAs, dateline rule |
| `CapabilityGrid.jsx` | 3 × 2 grid of capability blocks |
| `StatStrip.jsx` | Big-figure stat strip (assets under coverage, geographies, etc.) |
| `Coverage.jsx` | Two-column "where we cover" — geographies left, sectors right |
| `Footer.jsx` | Heavy ink footer with locale + legal + offices |

## What is intentionally missing

- Customer logos (would normally appear in a strip — we don't have permission to invent them)
- Press / awards section
- Long-form thought leadership articles
- Cookie / consent banner

## How to view

Open `index.html`. Scroll the page.
